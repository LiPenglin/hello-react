#before start 
## install nodejs

- [nodejs](https://nodejs.org/zh-cn/)
- extract
- add to path
- `node -v`

## create a react object

- `npm install -g create-react-app`
- `create-create-app hello-react`
- `cd hello-react`

## create index

- `src/index.css`
- `src/index.js`
- `npm start`
- `can view chessboard in the browser`
